# Deep Learning (CNN)

## About

This project was completed as part of the **Deep Learning (DeLearn)** module in the 
[Master of Science in Engineering](https://www.msengineering.ch/) (MSE) program. The goal is to explore Convolutional 
Neural Networks (CNNs) for image classification on the **iCoSimal V3** dataset, a balanced collection of 30,000 animal 
images across 10 categories (cat, chicken, cow, dog, elephant, horse, rabbit, sheep, squirrel, zebra). Experiments 
cover progressive CNN depth, hyperparameter tuning, regularization, and optimizer comparisons.

## Setup

Developed with **Python 3.12**. It is recommended to use a virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate
```

Then install dependencies with:
```bash
pip install -r requirements.txt
```
